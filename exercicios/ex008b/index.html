Segundo Jeff Noble, no seu libro HTML para Leigos:

Bem vindo às possibilidades desenfreadas, malucas e maravilhosas da World Wide Web, ou, mais, simplesmente, a web.HTML, XHTML e CSS Para Leigos, revela os detalhes das linguagens de marcação que são a veia da web – a Hypertext Markup Language (HTML) e sua prima, XHTM, junto com a linguagem da Cascading Style Sheet (CSS), usada para fazer com que outras coisas pareçam boas. HTML e XHTML (usamos (X)HTML neste livro para fazer referência às duas) e CSS são usadas para criar páginas web. Aprender a usá-las o coloca no grupo dos autores e desenvolvedores de conteúdo web. Lá tentou criar suas próprias páginas de web? Achou tudo muito proibitivo? Agora você pode relaxar! Se você consegue teclar os número em um telefone ou encontrar suas chaves de manhã, você também pode se tornar um autor (X)HTML. Sem brincadeira! Com linguajar técnico e português simples sobre o hipertexto, (X)HTML, e a web, incluímos diversos exemplos, mais instruções com tag por tag para ajudá-lo a criar suas páginas web com o mínimo estardalhaço. Fornecemos também exemplos sobre o que fazer com suas páginas web depois que elas forem criadas para que você possa publicá-las. Explicamos também as diferenças entre HTML4, HTML5 e XHTML, para que você possa decidir se quer ficar com a linguagem de marcação mais conhecida e mais antiga da web (HTML) ou sua sucessora mais recente e maior (XHTML).Pense nesse livro como um guia amigável e acessível para dominar (X)HTML E CSS!
